喜大普奔解决 hexo 数学公式下划线问题

Posted on 2018-05-11 |

数学公式困扰我很久，试过很多方法： - next 主题自带的，发现 mathjax 不好使， - hexo-renderer-kramed 不好使 - hexo-math 可以但是对\[ 无效，对\] $$有效 - 试了Katex 显示公式重复。 最后是 hexo-math 加 hexo-renderer-pandoc 有效的。 使用这个 renderer 之前请确保你已经安装了 Pandoc，然后卸载 Hexo 自带的 renderer，安装 pandoc renderer:

hexo 配置添加

主题配置文件添加：

如果你使用这款 Pandoc renderer，那么你书写 Markdown 时候需要遵循 Pandoc 对 Markdown 的规定。有一些比较明显的需要注意的事项：正常的文字后面如果跟的是 list, table 或者 quotation，文字后面需要空一行，如果不空行，这些环境将不能被 Pandoc renderer 正常渲染。
